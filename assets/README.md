# Assets

- `example-card.png` — the artwork the bundled target tracks. **Print it or open it
  on a second screen and point the app's camera at it** to see the effects.
- `video.mp4` — *(optional)* drop a clip here to use the 🎬 Video effect. Without it,
  the video plane shows a labelled placeholder.
- `model.glb` — *(optional)* a 3D model. Set `modelSrc: './assets/model.glb'` in
  `src/config.js` to replace the procedural butterfly.
